ID: T1137.006
Title: Office Application Startup:Add-ins
URL: https://attack.mitre.org/techniques/T1137/006
Adversaries may abuse Microsoft Office add-ins to obtain persistence on a compromised system. Office add-ins can be used to add functionality to Office programs. [1] There are different types of add-ins that can be used by the various Office products; including Word/Excel add-in Libraries (WLL/XLL), VBA add-ins, Office Component Object Model (COM) add-ins, automation add-ins, VBA Editor (VBE), Visual Studio Tools for Office (VSTO) add-ins, and Outlook add-ins. [2] [3] Add-ins can be used to obtain persistence because they can be set to execute code when an Office application starts.
